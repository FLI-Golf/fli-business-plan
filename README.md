# FLI Business Plan

A comprehensive business planning application built with modern web technologies.

## Core Technologies

- SvelteKit 2.x - Fast and efficient web framework
- PocketBase - Backend and authentication
- TailwindCSS - Utility-first styling
- TypeScript - Type-safe development
- Bits UI - Modern UI component library
- SuperForms - Form handling and validation
- Zod - Schema validation
- Lucide Icons - Beautiful SVG icons

## Getting Started

1. Clone the repository:

```bash
git clone [your-repo-url]
cd fli-business-plan
```
